SQL - Structured Query Language

- Used to interact with relational databases.
- Many flavors: MySQL, Micrososft SQL Server, Amazon Redshift, Oracle DB

Find data:
SELECT (column_1, column_2, ...) FROM table [WHERE condition GROUP BY columns LIMIT number];


Add data:
INSERT INTO table (column_1, column_2, ...) VALUES (value_1, value_2, ...);


Update data:
UPDATE table SET column_1=value_1, column2=value_2, ... WHERE condition;


Delete data:
DELETE FROM table WHERE condition;
